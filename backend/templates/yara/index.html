ScriptObservatory Yara Alerting

Yara support is still experimental, but you can now be alerted whenever a resource is observed that matches a given Yara rule.

To start using this feature, your email address needs to be on the list of approved addresses. If you'd like to be added, send me an email.

If you'd like to remove a previously-submitted Yara rule, click here.

The alerting system uses a "token bucket" scheme to prevent a surge of matches from overloading your mailbox with emails. When you add a rule, it starts with 5 tokens. Each time an observed resource matches the rule, a token is taken away before an email is sent. Tokens are replenished at a rate of one token per minute, and each rule can have a maximum of 10 tokens at one time. If the token balance ever reaches zero, the rule is automatically removed from the database.

If a rule's balance ever reaches zero, you'll receive the same notification email you would have received if you had manually removed it by submitting the removal code.

If you're already registered, submit a new rule by filling out the form below.
